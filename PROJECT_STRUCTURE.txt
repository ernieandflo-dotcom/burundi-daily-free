═══════════════════════════════════════════════════════════════
  MISE EN PLACE (5 étapes, ~15 minutes)
═══════════════════════════════════════════════════════════════

  1. Créer un compte gratuit sur huggingface.co
     → huggingface.co/settings/tokens → New token (rôle : Read)
     → Copiez votre token (commence par hf_...)

  2. Créer un dépôt GitHub PUBLIC nommé "burundi-daily-free"

  3. Uploader tous ces fichiers dans ce dépôt

  4. Activer GitHub Pages :
     Settings → Pages → Source : GitHub Actions

  5. Ajouter votre clé HF dans GitHub Secrets :
     Settings → Secrets and variables → Actions → New secret
       • HF_API_KEY → votre token Hugging Face

  Déclencher manuellement la première fois :
     Actions → "Burundi Daily Intelligence (Free)" → Run workflow

  ✅ URL mobile : https://VOTRE_USERNAME.github.io/burundi-daily-free/
